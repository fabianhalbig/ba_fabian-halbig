Features von Course-Planner

Erstellung und abonnieren von Courses

Jeder User kann einen Course erstellen, in dem zu erledigenden Tasks mit Deadline hinzugefügt werden können.

Alle Courses können von Usern abonniert werden, um die Tasks auf einem Blick zu sehen.

Taskübersicht zu jedem Course

Die Tasks eines Users können pro Course angezeigt werden oder alle bereits bearbeitete oder erledigte Tasks.

Damit hat jeder User einen Überblick auf die anstehenden Aufgaben und die damit verbundenen Deadlines.

Tracking der abzuarbeitenden Tasks

Zu jedem Task von einem User wird ein Status festgehalten, der von dem User von "Offen" über "In arbeit" auf "Fertig" gesetzt werde kann.

Dadurch hat der Course-Creator eine Übersicht, wie weit die Abonnenten mit der abarbeitung der gestellten Tasks voranschreiten, um frühzeitig einer Auffschiebung entegegenzuwirken.
